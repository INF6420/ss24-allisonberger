INF 6120

Types of Reference Services in a Public Library

This essay is is an integrated report about several of my observations at a suburban public library. It focuses on the different types of reference services available in public libraries. There is a special concentration on three types of reference services: readers’ advisory, ready reference, and literacy programs, given their frequent occurrence during my observation period.

INF 7996

The Relationship Between Social Media Presence and Library Usage in Public Libraries

In recent years, many public libraries have begun using social media as a marketing tool as a way to increase library usage. To prove this action is valid, the purpose of this study is to determine the relationships between a public library’s social media presence and its in-person event attendance and book circulation.
